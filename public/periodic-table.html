Interactive Periodic Table of Trends

Instead of memorizing, let's visualize! Select a periodic trend to see how it changes across the table. The color scale shows the intensity: Red is for low values and Blue is for high values.

  • Click a trend button below (e.g., Atomic Radius).
  • Observe the color pattern and the trend arrows that appear. Use the scrollbar if needed.
  • Click on any element to see its name, symbol, and exact value for the selected trend.
